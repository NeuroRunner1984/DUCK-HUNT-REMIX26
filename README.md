# DUCK-HUNT-REMIX26
A demo of a classic NES video game, showcasing my ability to primarily use JavaScript, HTML-5, and CSS.  

This demo game is a "REMIX" of a game I made that was initially made following a tutorial that be found below:

# [Duck Hunt](https://youtu.be/YGBi4lR0swY)
- Coding Tutorial: [https://youtu.be/YGBi4lR0swY](https://youtu.be/YGBi4lR0swY)
- Demo: https://imkennyyip.github.io/duck-hunt/

 It was made to primarily showcase my ability to use javascript proficiently. Along the way I learned how to use HTML5 canvas.
With that said, this is "The Remix", not the regular Duck Hunt game, nor a complete copy of the tutorial, it is something all my own, made within the bounds of respecting the legal I.P. of original Nintendo Duck Hunt game which was an early video game staple for me growing up in the 90's. Lastly, it was necessary to code in order to prove to myself that I actually learned something & that I hadn't just copied a tutorial. The use of A-I in this demonstration is MINIMAL and was only used from Google's A-I summarized search results (which are default these days) and GitHub Copilot that is attached to VS Code. 

What the "Remix" includes(not included in initial demo or game): 
- Original self-produced background musical soundtrack
- Self designed background
- Responsive design
- A Start Game overlay & function(s)
- A Game Over overlay & function(s)
- A Restart Game Button & function(s)
- A functional timer & score
- Additional sound effects
